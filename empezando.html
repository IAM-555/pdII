HTML y HTML5

HTML es el lenguaje más usado para construcción de páginas Web. La sigla HTML significa Hyper Text Markup Language (lenguaje de marcas de hipertexto). HTML no es un lenguaje de programación, sino de marcas con sentido. Llamamos a dichas marcas etiquetas. Éstas le indican al navegador qué es lo que debe mostrar en pantalla.


¿Cuál es el origen de HTML5?

El lenguaje HTML5 se desarrolló con la cooperación de dos entidades, World Wide Web Consortium (W3C) y Web Hypertext Application Technology Working Group (WHATWG). WHATWG estaba trabajando en mejoras para los formularios en la web, mientras que W3C trabajaba en XHTML2.0. En el año 2006 decidieron juntarse a trabajar en una nueva y mejor versión del lenguaje HTML, dando así origen a la era de HTML5

Reglas basicas de HTML5

  • Las nuevas funcionalidades se basan en HTML, CSS, DOM, y JavaScript.
  • Reduce el uso de plugins externos, por ejemplo: Flash Player.
  • Mejora el manejo de errores.
  • Tiene mejores etiquetas de marcación.
  • HTML5 es independiente del dispositivo de visualización.
  • El proceso de desarrollo es visible para el público.

Etiquetas

Las etiquetas son los elementos que componen el HTML. Cada etiqueta tiene un significado y una función específica. Las etiquetas se escriben entre corchetes angulares (<>) y generalmente vienen en pares, es decir, una etiqueta de apertura y otra de cierre.

Roles

Los roles son atributos que se pueden agregar a las etiquetas HTML para indicar su función o propósito en la página web. Los roles ayudan a los navegadores y a los lectores de pantalla a entender mejor la estructura y el contenido de la página.



CSS3

Que es CSS?

CSS (Cascading Style Sheets) u Hojas de estilo en cascada en español, no es un lenguaje de programación, pero tampoco es un lenguaje de marcado. Es un lenguaje de hojas de estilo, es decir, te permite aplicar estilos a elementos en documentos HTML.

Sintaxis y reglas de estilo

  • El selector es el elemento sobre el cual se aplica la regla de estilo. Por ejemplo: body, h1, section, p
  • La propiedad es la característica que se modifica, por ejemplo, color, tipografía, tamaño, etc.
  • El valor que se aplica a la propiedad, por ejemplo: red, arial, 12px, etc.

Especificidad

La especificidad es un concepto importante en CSS que determina qué reglas de estilo se aplican a un elemento en particular. La especificidad se basa en la combinación de selectores utilizados para seleccionar un elemento.


Si cambió estilos sobre algún elemento este tomará el que tenga mayor especificidad
Etiqueta HTML unidades 0,0,1
Clase CSS decenas 0,1,0
Id CSS centenas 1,0,0
Selector universal miles 0,0,0
!important sobre todas No se recomienda

Shorthand

El "shorthand" (abreviatura) en CSS es una técnica que permite definir varias propiedades relacionadas en una sola línea de código, lo que hace que el código sea más conciso y fácil de mantener.

Ejemplos comunes de propiedades shorthand en CSS:

  1. margin/padding: En lugar de definir cada lado por separado.

    2. css
    /* Forma larga */
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 10px;
    margin-left: 20px;

    /* Shorthand */
    margin: 10px 20px 10px 20px; /* arriba derecha abajo izquierda */
    margin: 10px 20px; /* arriba/abajo izquierda/derecha */
  2. border: Combina width, style y color.

    3. css
    /* Forma larga */
    border-width: 1px;
    border-style: solid;
    border-color: black;

    /* Shorthand */
    border: 1px solid black;
  3. font: Combina múltiples propiedades de texto.

    4. css
    /* Forma larga */
    font-style: italic;
    font-weight: bold;
    font-size: 16px;
    line-height: 1.5;
    font-family: Arial, sans-serif;

    /* Shorthand */
    font: italic bold 16px/1.5 Arial, sans-serif;
  4. background: Para múltiples propiedades de fondo

    5. css
    /* Forma larga */
    background-color: white;
    background-image: url('imagen.jpg');
    background-repeat: no-repeat;
    background-position: center;

    /* Shorthand */
    background: white url('imagen.jpg') no-repeat center;
